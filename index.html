Portfolio

UConsole

UConsole is a simple free in-game console which you can add to your Unity Projects to help you debug and test out custom commands during development.
UConsole was heavily inspired by the console which is in Unreal Engine 4, I felt that people could have a use for this in the projects as well in games built in the Unity Engine. All it is is one prefab which you just have to drag into your scene then it starts to work. It relies on TextMeshPro's text to display clear and customizable text in the console window, it was created on Unity 2018.3 with the new prefab system.

Block Factory

Block Factory is a sandbox factory builder where you are tasked to build an efficient factory to sell and earn as much money as you can.
Block Factory was created for my final unit in my first year in Level 3 Games Development at Northbrook, where we were set a theme of revolution then got given 3 weeks to create a game. All my assets were created in MagiaVoxel and the game was created in Unity. This was a hard project from the logical sides of things, mainly getting the pipes to work with the items flowing long them was a big challenge, I had to make sure the pipes checked if a new pipe had been placed on an open side and tell the information to items which were going through, but still making it optimised when there were large amounts of pipes placed.

Prototypes

VR Physics Gun

This prototype was my first tests at creating a VR game. The idea was to recreate the physics gun from Garry's Mod. If you haven't played Garry's Mod, the physics gun is a weapon which has the power to pick up and move items at the end of its ray. In this prototype, I used a primitive cube as the weapon, which could be held and thrown in any hand. When you press the trigger a small straight laser would show up at the end of the weapon and you could pick up items if they had a certain script on them. As you can see in the video, items aren't fixed at the end of the ray, which is similar to the ray in Garry's Mod. However, I couldn't manage to make a ray render, so I had to stick to my straight line laser. This prototype just took a day to create and used the SteamVr Plugin for Unity and the primitive cubes for the assets.

VR Physics Gun on Github

BattleShips

Battleships is a multiplayer top-down pirate game where your goal is to sink the most amount of ships without getting sunk by other players. This was done as apart of a college assignment where I documented the whole process of development here under unit 12. The goal of this prototype was just to learn more about network programming in Unity, things I had to think about during planning, how to overcome problems and good ways to write network code. I did learn a lot from creating this, even though there are lots of problems with the code, I have noticed them and understand that there are now other ways to achieve the same thing. Just had to get my first messy attempt at networking out of the way and start learning.
Battleships was created in Unity 2018.2.14 using Forge Networking Remastered and Photoshop for the graphics.
